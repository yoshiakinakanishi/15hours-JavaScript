# 15hours-JavaScript

10時限目のendファイルをローカルに落として
npm installでnode modulesを入れて
cd移動してnpm startを実行する
localhost3000を入力で立ち上げて、
コマンドラインで操作のたびに、下記のようなCRUDが実行される
いったんブラウザを終了してから、再度localhost3000にアクセスすると
前回の情報が正しく保存されている！！
